## Как скомпилировать документацию

### Установка Typst

**Linux/macOS:**
```bash
# Через Cargo (рекомендуется)
cargo install --git https://github.com/typst/typst --locked typst-cli

# Или скачайте бинарник с GitHub Releases
```

**Windows:**
```powershell
# Через Scoop
scoop install typst
```

### Компиляция

```bash
# Компилировать один раз
typst compile README.typ README.pdf

# Автоматическая перекомпиляция при изменениях
typst watch README.typ README.pdf
```

## Изменения в конвертации

- LaTeX команды заменены на Typst синтаксис
- `\section` → `=`, `\subsection` → `==`, `\subsubsection` → `===`
- `\begin{enumerate}` → нумерованные списки с `+`
- `\begin{itemize}` → маркированные списки с `-`
- `\textbf{text}` → `*text*`
- `\texttt{code}` → `` `code` ``
- `\begin{verbatim}` → блоки кода с ` ``` `
- Таблицы конвертированы в Typst формат с использованием `#figure()`
- Изображения конвертированы в `#figure(image(...))`

## Что делать с README.txt
