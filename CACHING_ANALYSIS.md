### Используемая структура данных
Проект использует **двусвязный список (doubly-linked list)** для хранения заметок в оперативной памяти:

```cpp
struct NoteNode {
    Note data;                   // Данные заметки
    NoteNode* next;              // Указатель на следующий узел
    NoteNode* prev;              // Указатель на предыдущий узел
};
```

Класс `NoteManager` управляет этим списком:
```cpp
class NoteManager {
private:
    NoteNode* head;             // Голова списка
    NoteNode* tail;             // Хвост списка
    int noteCount;              // Текущее количество заметок
    int nextId;                 // Следующий доступный ID
    // ...
};
```

### Хранение данных
- **В памяти**: Двусвязный список узлов (NoteNode)
- **На диске**: 
  - Файл метаданных `notes_metadata.dat` содержит информацию о всех заметках
  - Отдельные текстовые файлы для содержимого каждой заметки в директории `notes/`

### Загрузка данных
Заметки загружаются из файла метаданных в список при запуске программы (`NoteManager::loadFromFile()`), но это не является кешированием - это просто основная структура данных программы.
